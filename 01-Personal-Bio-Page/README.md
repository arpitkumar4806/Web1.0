# Personal Bio Page

A minimal personal bio webpage built with plain HTML and CSS. This repository contains a small single-page site that showcases a profile, short bio, and links to social profiles.

## Features

-   This project is a simple portfolio/homepage to personalize and expand with projects and contact details.
-   Provides a clean and intuitive user interface.
-   Responsive design that works well on desktop and mobile devices.

## Files

-   `index.html` — The single-page HTML layout for the personal bio.
-   `style.css` — Styles for layout, typography, and the profile illustration.
